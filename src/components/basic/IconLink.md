### Default Link Button

```jsx
<div style={{ padding: '10px'}}>
	<IconLink
		icon="fa fa-users"
		size="large"
		description="Visit our users dashboard to learn more."
		text="Default"
		href="https://www.w3schools.com" />
	<IconLink
		className="hovered"
		icon="fa fa-users"
		size="large"
		description="Visit our users dashboard to learn more."
		text="Hovered"
		href="https://www.w3schools.com" />
	<IconLink
		className="active"
		icon="fa fa-users"
		size="large"
		description="Visit our users dashboard to learn more."
		text="Active"
		href="https://www.w3schools.com" />
	<IconLink
		className="focused"
		icon="fa fa-users"
		size="large"
		description="Visit our users dashboard to learn more."
		text="Focused"
		href="https://www.w3schools.com" />
	<IconLink
		className="disabled"
		icon="fa fa-users"
		size="large"
		description="Visit our users dashboard to learn more."
		text="Disabled"
		href="https://www.w3schools.com" />
</div>
```

Html Snippet:

```html
<a href="https://www.w3schools.com" class="dg_icon-link dg_icon-link--large" aria-label="Visit our users dashboard to learn more.">
	<i aria-hidden="true" class="dg_icon fa fa-users"></i>
	<span>Default</span>
</a>
```
