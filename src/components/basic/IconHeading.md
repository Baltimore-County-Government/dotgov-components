### Default Icon Heading

```jsx
<section  style={{ padding: '15px' }}>
    <IconHeading text="Latest Headlines" icon="fas fa-star" />
</section>
```

### Icon Heading on a Light Background

```jsx
<section class="dg_section light" style={{ padding: '15px' }}>
    <IconHeading text="Latest Headlines" icon="fas fa-star" />
</section>
```
